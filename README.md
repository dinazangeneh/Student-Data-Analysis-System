# Student Data Analysis System 📊

A Python-based application for analyzing student data from Excel files.  
The project uses **Pandas** for data processing and **Matplotlib** for data visualization.

The system can analyze student information, generate statistical reports, and create visual charts.

---

## Features

- 📂 Read student data from Excel files
- 🔍 Automatic detection of column names
- 📊 Statistical analysis of student data
- 👥 Gender-based analysis
- 🎓 Age-based score analysis
- 📈 Data visualization using charts
- 📄 Export analysis reports to Excel

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- OpenPyXL

---
## How to Run

1. Make sure Python 3 is installed.
2. Run the following command:

```bash
python Student Data Analysis.py
```
